Angular.js

Es un framework para aplicaciones web, es mantenido por Google y desarollaodres independientes y esta orientado al desarrollo de SPA (Single page applications)


SPA

SPA o Single-page application es un enfoque usado para ofrecer al usuario una mejor experiencia de uso a la hora de interactuar con una aplicación web, la cual permite que el usuario lleve a cabo un proceso de navegación fluido puesto que la aplicación carga los recursos principales al inicio y luego carga de manera dynamica lo que le haga falta.

Caracteristicas

  • Usa el patrón arquitectural MVC
  • Sistema de templates muy similar a Mustache
  • Hace uso de directivas inmersas en el html
  • Permite crear directivas propias
  • Double way binding
  • Ofrece librerias para extender funcionalidades
    • $http
    • $window
    • $animate
    • etc...

Ventajas

  • Permite realizar aplicaciónes en muy poco tiempo
  • Gran comunidad
  • Bien documentado
  • Mantenido por Google

Desventajas

  • (Para algunos) Más complejo que otros frameworks
  • Abtracción / Control
